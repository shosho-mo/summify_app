🚀 Summify | AI-Powered Book Summaries
Summify is a sophisticated mobile application built with Flutter, designed to provide smart, concise summaries of international books in future-oriented specializations. The app bridges the gap between complex global knowledge and accessible learning through cutting-edge AI integration.

✨ Key Features
AI-Generated Summaries: High-quality summaries produced by a blend of advanced AI tools and expert human oversight.

Intelligent Chatbot: An interactive AI assistant powered by OpenAI to discuss book insights and answer user queries in real-time.

Modern Backend: Leveraging Supabase for secure authentication, real-time database management, and scalable cloud storage.

Seamless UI/UX: A sleek, user-centric design that ensures a smooth and engaging learning journey.

Cross-Platform Performance: High-performance experience across both Android and iOS devices.

🛠️ Tech Stack
Frontend: Flutter & Dart.

Backend as a Service: Supabase (Auth, Database, Storage).

AI Engine: OpenAI API for the smart chatbot interface.

State Management: Built with Bloc/Provider for robust and maintainable state handling.

Architecture: Adhering to Clean Architecture principles for scalability and testability.

🚀 Getting Started
To run this project locally:

Clone the repository:

Bash
git clone https://github.com/shosho-mo/summify_app.git
Install dependencies:

Bash
flutter pub get
Setup Environment:

Create a .env file and add your Supabase and OpenAI credentials.

Launch the application:

Bash
flutter run
